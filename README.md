500px-to-kde-wallpaper
======================

This bash script will take any 500px URL and create a KDE wallpaper folder structure with the right meta data.

Prerequisites
-------------

* curl
* wget
* ImageMagick
* grep (with perl regex capabilities)
* sed

How to use?
-----------

./wallpaper500px.sh "[URL]"
